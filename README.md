Dataset (Taken from the links below) - 
  1. Edited Images - https://www.kaggle.com/datasets/divg07/casia-20-image-tampering-detection-dataset
  2. AI Images - https://www.kaggle.com/datasets/birdy654/cifake-real-and-ai-generated-synthetic-images
  3. Real Images - https://www.kaggle.com/datasets/birdy654/cifake-real-and-ai-generated-synthetic-images

For Replicating this project - 
  1. Copy all the files from this Repo in the exact same format as given
  2. Create a new folder for the dataset and download the files from the links above
  3. Create a Folder named "Processed Data" and divide it into 3 sub folders
     - Test
     - Train
     - Val
     Each with sub folders - AI, Real and Edited comprising small parts of the data, so that it could be properly Tested, Trained and Validated
